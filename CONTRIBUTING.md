## Contributor License Agreement (CLA)

All contributors must agree to the Contributor License Agreement (CLA)
before their contributions can be accepted.

By opening a pull request, you confirm that you have read and agree
to the CLA:

See CLA.md in this repository.

Contributions without agreement to the CLA cannot be merged.

## Code Standards

- Keep changes focused and minimal.
- Provide clear commit messages.
- Include tests when applicable.
- Avoid introducing unnecessary dependencies.
